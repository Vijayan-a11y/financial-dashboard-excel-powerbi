This project demonstrates a Financial Dashboard created using Excel and Power BI.

- Region-wise and Category-wise analysis
- Interactive filters and visuals

## 🛠 Tools Used
- Microsoft Excel
- Power BI

## 📌 Insights
- Identified top-performing categories
- Analyzed profit trends
